# 🎯 Project Vision

> **Bukan sekadar bot apply LinkedIn, tapi end-to-end career operations platform** —
> discover → score → tailor → apply → track → analyze → follow-up — multi-platform,
> multi-profile, multi-tenant, runs locally first, deployable to cloud later.

## 🌟 North Star
Build a **production-grade, privacy-first, plugin-based, AI-augmented Python platform**
that automates the entire job application lifecycle.

## 🎯 Primary Goals
1. **Eliminate Repetitive Apply Work** — automate 80%+ Easy Apply forms across major platforms
2. **Increase Application Quality** — AI-tailored resume, cover letter, fit scoring
3. **Pipeline Visibility** — ghosting detector, health score
4. **Career Outcomes** — interview prep, follow-up, company research

## 🚫 Non-Goals
- Lie on applications
- Bypass CAPTCHAs aggressively
- Apply > 100/day (suspension risk)
- Replace human judgment for high-priority jobs

## 🧭 Design Principles
1. **Privacy First** — all local, no telemetry
2. **Plugin Architecture** — drop file → new platform
3. **Honest AI** — anti-hallucination non-negotiable
4. **Graceful Degradation** — never crash
5. **Resumable Workflows** — Pause/Resume/Stop respected
6. **Observable** — every action logged
7. **Configuration over Code** — YAML-driven
8. **Multi-Language Native** — global from day one

## 📐 Success Metrics
| Metric | Target |
|---|---|
| Apply success rate | ≥ 90% |
| Questions auto-answered | ≥ 95% |
| Daily applies safe | ≥ 20 |
| AI hallucinations | 0 |
| Setup time | < 15 min |
